Privacy Policy for AI Image Studio

Last Updated: July 5, 2025

Overview

AI Image Studio ("the Extension") is committed to protecting your privacy. This privacy policy explains how we handle information when you use our Chrome extension.

Information Collection and Use

What We DON'T Collect

AI Image Studio does NOT collect, store, or transmit any personal information, including but not limited to:

  • Browsing history
  • Personal data
  • Image content
  • API keys (stored locally only)
  • Usage analytics
  • Cookies or tracking data

Local Storage Only

The Extension stores the following data locally on your device using Chrome's storage API:

  • Your API keys (encrypted locally)
  • Extension settings (provider preferences, download options)
  • Temporary image data during processing (cleared after use)

This data never leaves your device except when:

  • Images are sent directly to AI providers (OpenAI or Google) for processing
  • Images are sent to PhotoRoom API for background removal (if enabled)

Third-Party Services

The Extension communicates directly with:

  1. OpenAI API (if configured)
    Purpose: Image generation using GPT Image 1
    Data sent: Cropped image and generation parameters
    Privacy Policy
  2. Google AI API (if configured)
    Purpose: Image generation using Imagen
    Data sent: Cropped image and generation parameters
    Privacy Policy
  3. PhotoRoom API (optional)
    Purpose: Background removal
    Data sent: Generated image for background removal
    Privacy Policy
  4. ExtensionPay (for subscription management)
    Purpose: Trial and subscription verification
    Data sent: Extension ID and user identifier
    Privacy Policy

Data Security

  • API keys are stored using Chrome's storage.sync API with local encryption
  • All API communications use HTTPS encryption
  • No data is stored on external servers
  • Image processing happens directly between your browser and the AI provider

Permissions Explained

The Extension requires these permissions:

  • storage: Save your settings and API keys locally
  • activeTab: Access images on the current webpage you're viewing
  • contextMenus: Add right-click menu option for images
  • tabs: Open settings and payment pages
  • downloads: Save generated images to your computer
  • host permissions: Access AI provider APIs and images on websites

Data Retention

  • Temporary image data is cleared immediately after processing
  • Settings and API keys remain until you uninstall the extension
  • No logs or history of your image generations are kept

Children's Privacy

The Extension is not intended for use by children under 13 years of age. We do not knowingly collect information from children under 13.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected in the "Last Updated" date above.

Your Rights

  • Delete all local data by uninstalling the extension
  • View stored data in Chrome's extension settings
  • Revoke API access by removing your API keys
